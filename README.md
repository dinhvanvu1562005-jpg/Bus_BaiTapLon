# Bus Bai Tap Lon

## Giới thiệu
Hệ thống quản lý hiến máu (BloodLink) viết bằng PHP.

## Chức năng
- Đăng nhập admin
- Dashboard quản lý
- Authentication system
- Session login

## Công nghệ sử dụng
- PHP
- MySQL
- HTML/CSS
- XAMPP

## Cách chạy project
1. Clone repo
2. Copy vào htdocs
3. Start Apache + MySQL
4. Truy cập localhost
